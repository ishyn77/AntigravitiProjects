# adfil todo

Это простое веб-приложение для управления списком задач (ToDo), созданное с использованием Flask и Bootstrap.

## Особенности

- **Добавление задач**: Вы можете легко добавлять новые задачи в свой список.
- **Удаление задач**: Удаляйте задачи, когда они вам больше не нужны.
- **Отметка о выполнении**: Переключайте статус задачи между "выполнено" и "не выполнено".
- **База данных SQLite**: Все ваши задачи сохраняются локально в базе данных SQLite.
- **Адаптивный дизайн**: Благодаря Bootstrap приложение отлично выглядит как на компьютерах, так и на мобильных устройствах.

## Стек технологий

- **Backend**: Python, Flask
- **ORM**: Flask-SQLAlchemy
- **База данных**: SQLite
- **Frontend**: HTML, CSS, Bootstrap

## Установка и запуск

Для запуска этого проекта на вашем локальном компьютере выполните следующие шаги:

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd flask-todo-app
```

### 2. Создание и активация виртуального окружения

На Windows:
```bash
python -m venv .venv
.venv\Scripts\activate
```

На macOS/Linux:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

```bash
python app.py
```

После запуска приложение будет доступно по адресу: `http://127.0.0.1:5000/`

## Структура проекта

- `app.py`: Основной файл приложения Flask с описанием моделей базы данных и маршрутов.
- `requirements.txt`: Список необходимых библиотек Python.
- `templates/`: Папка с HTML-шаблонами (напр. `index.html`).
- `static/`: Папка для статических файлов (CSS, JS, изображения).
- `todo.db`: Файл базы данных SQLite (создается автоматически при первом запуске).
